LangGraph Supervisor is a Python library for creating **hierarchical multi-agent systems** built on top of [LangGraph](https://github.com/langchain-ai/langgraph). A supervisor agent coordinates multiple specialized sub-agents, controlling all communication flow and task delegation.

## Key features

<CardGroup cols={2}>
  <Card title="Supervisor orchestration" icon="sitemap">
    A central supervisor agent delegates to specialized sub-agents using automatically created handoff tools.
  </Card>
  <Card title="Flexible message history" icon="message">
    Control context with `full_history` or `last_message` output modes.
  </Card>
  <Card title="Multi-level hierarchies" icon="diagram-project">
    Compose supervisors into nested teams — supervisors managing supervisors.
  </Card>
  <Card title="Built-in memory" icon="database">
    Add short-term and long-term memory by passing a checkpointer or store to `.compile()`.
  </Card>
</CardGroup>

## Installation

```bash
pip install langgraph-supervisor
```

Requires Python >= 3.10.

## Quick example

```python
from langchain_openai import ChatOpenAI
from langgraph_supervisor import create_supervisor
from langgraph.prebuilt import create_react_agent

model = ChatOpenAI(model="gpt-4o")

math_agent = create_react_agent(model=model, tools=[add, multiply], name="math_expert")
research_agent = create_react_agent(model=model, tools=[web_search], name="research_expert")

workflow = create_supervisor(
    [research_agent, math_agent],
    model=model,
    prompt="Use math_expert for calculations, research_expert for facts."
)

app = workflow.compile()
result = app.invoke({"messages": [{"role": "user", "content": "What is 2 + 2?"}]})
```
